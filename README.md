# PokePlus
PokePlus is an automation script using Ankulua like Pokemon GO Plus.

## Environment
- OS: Android
- Automation Environment: Ankulua
- Language: Lua
- Ankulua Website:
English: http://ankulua.boards.net/; 
Chinese: http://ankulua-tw.boards.net/

## Features:
Blind Click, then
- Catch Pokemon.
- Spin Pokestop.
